Google Sheets Setup

All property data is stored securely in your own Google Sheet. Configure once — works on all devices. Your data never touches any third-party server.
1
Go to Google Cloud Console → New Project → Enable Google Sheets API
2
Go to Credentials → Create OAuth 2.0 Client ID (Web Application type). Under Authorized JavaScript Origins, add your GitHub Pages URL: https://yourusername.github.io
3
Create a new Google Sheet. Copy the Sheet ID from the URL — it's the long string between /d/ and /edit
